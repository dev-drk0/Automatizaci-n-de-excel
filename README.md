# Sistema de automatización administrativa
--------------------------------

Tecnologías:

- Python
- Flet
- Pandas
- Openpyxl
- PDFPlumber
- Matplotlib

Características:

- procesamiento masivo de PDFs
- extracción automática de datos
- generación de Excel
- generación de gráficas
- estadísticas automáticas
- interfaz gráfica
- soporte multiplataforma

Resultados:

- 41 facturas procesadas
- exportación automática
- pruebas realizadas en:
    - macOS
    - Ubuntu Linux

automatiza el procesamiento de lotes de facturas y genera reportes administrativos en Excel de manera automática.
